## About

This repository contains the source code for the Pomelo documentation website and landing page. It is built using **[Astro](https://astro.build/)** and designed to be fast, responsive, and easy to maintain.

## Local Development

To run the documentation site locally:

1. **Install dependencies**
   ```bash
   pnpm install
   ```

2. **Set up environment variables**  
   Copy `.env.example` to `.env` if required for local dev.

3. **Start the development server**
   ```bash
   pnpm dev
   ```

4. **Build for production**
   ```bash
   pnpm build
   ```
