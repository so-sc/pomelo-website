## Bun CLI

The admin directory contains two Bun-based tools:

- `admin/src/pomelo.ts` — the CLI tool for managing contests
- `admin/src/pomelod.ts` — the background daemon process

<Note>
  Both files use Bun shebangs and require Bun to be installed. If Bun is missing, install it with `curl -fsSL https://bun.sh/install | bash`.
</Note>
